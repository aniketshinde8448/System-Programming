Input Assembly File: add2.asm

Assembler python program: assembler.py

Main Executing Program: main.py

Pre-requisite input files required for the assembler program: 	1. instr.txt
								2. reg.txt
								3. mod.txt
								4. opsymtab.txt

Steps to run the Assembler program:

1. python3 main.py add2.asm [options : -s/-l/-i/-lst/-obj]

options: 1. -s : To Display Symbol Table.
	 2. -l : To Display Literal Table.
	 3. -i : To Display Intermediate Code.
 	 4. -lst: To Display .lst file.
	 5. -obj: To Display Object Code file.
	 

OUTPUT:

Output will be displayed according to the selected option.
